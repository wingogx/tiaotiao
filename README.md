# 跳一跳小游戏

一个基于微信小游戏平台开发的跳一跳游戏。

## 游戏特色

- 🎮 简单易上手的操作方式
- 🏆 分数系统和最高分记录
- 🎵 音效和震动反馈
- 🌈 精美的渐变背景和云朵装饰
- 📱 完美适配微信小游戏平台

## 游戏玩法

1. **开始游戏**：点击"开始游戏"按钮
2. **控制角色**：长按屏幕蓄力，松开手指跳跃
3. **跳跃技巧**：蓄力时间越长，跳跃距离越远
4. **得分规则**：
   - 普通平台：10分
   - 特殊平台：20分
   - 奖励平台：50分
5. **游戏结束**：角色掉出屏幕即游戏结束

## 开发说明

本游戏使用微信小游戏原生API开发，无需额外的游戏引擎。代码结构清晰，易于理解和扩展。

### 主要API使用：
- `wx.createCanvas()` - 创建画布
- `wx.onTouchStart/Move/End()` - 触摸事件处理
- `wx.createInnerAudioContext()` - 音频播放
- `wx.vibrateShort()` - 震动反馈
- `wx.getStorageSync/setStorageSync()` - 本地存储
